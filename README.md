# 🌤️ Dashboard de Clima

Panel administrativo con datos meteorológicos en tiempo real, construido con React y Tailwind CSS. Permite buscar cualquier ciudad, ver su clima actual y consultar la tendencia de temperatura de los últimos 7 días.

🔗 **Demo en vivo:** https://dashboard-clima-murex.vercel.app/

## Features

- 🔍 Búsqueda y agregado de ciudades con geocodificación automática
- 📊 Tabla filtrable con datos meteorológicos por ciudad
- 📈 Gráfico de tendencia de temperatura (7 días) con Recharts
- 🌗 Modo oscuro / claro
- ⏳ Loading skeleton y manejo de errores de la API
- 📱 Diseño responsive (sidebar + navbar adaptables)

## Stack

- **React** (Vite)
- **Tailwind CSS**
- **Recharts** — visualización de datos
- **Axios** — consumo de API
- **React Icons**
- **[Open-Meteo API](https://open-meteo.com/)** — datos meteorológicos y geocodificación, gratuita y sin API key

## Correr el proyecto localmente

```bash
git clone https://github.com/francisco-martinez-dev/dashboard-clima.git
cd dashboard-clima
npm install
npm run dev
```
